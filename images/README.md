# Images and Visualizations

Store your images and visualizations in this folder.

## Suggested Images to Create

### 1. architecture.png
System architecture diagram showing the complete pipeline:
- Data collection → Processing → Fine-tuning → Evaluation

### 2. dataset_composition.png
Pie chart or bar chart showing:
- Synthetic data: 9,775 QA pairs (60.7%)
- Interview data: 6,338 QA pairs (39.3%)

### 3. evaluation_metrics.png
Infographic showing the 7 therapeutic evaluation metrics:
1. Active Listening
2. Empathy & Validation
3. Safety & Trustworthiness
4. Open-mindedness & Non-judgment
5. Clarity & Encouragement
6. Boundaries & Ethical
7. Holistic Approach

### 4. results_comparison.png
Bar chart comparing:
- Base models vs Fine-tuned models
- Different training configurations (synthetic, interview, combined)

### 5. topic_distribution.png
Chart showing the 33 mental health topics and their distribution

### 6. data_pipeline.png
Flowchart showing:
- Clinical transcripts → Local Mistral paraphrasing → Interview QA pairs
- GPT-3.5 generation → Synthetic QA pairs
- Combined → MentalChat16K

---

## Tools for Creating Visualizations

- **Python**: matplotlib, seaborn, plotly
- **Online**: Canva, Lucidchart, draw.io
- **Presentation**: PowerPoint, Google Slides

## Image Guidelines

- Use PNG format for clarity
- Maintain consistent color scheme
- Include clear labels and legends
- Cite source if using figures from the paper
